# Duel

[Duel](https://www.duelonline.sk/duelonline_hra.html) is an online quiz game from slovak national television. It consists of 10 multiple-choice questions with a 60-second time limit.

The difficulty for LLMs lies in the knowledge of obscure slovak words, annecdotes, and cultural references. Hence, the game is a great benchmark for testing the knowledge of LLMs in a language that is underrepresented in the training data.
