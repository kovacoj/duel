Duel — LLM Benchmark

Benchmarking LLMs against the RTVS Duel quiz — live automation + replayable runs.

What

Duel evaluates language models on a Slovak trivia quiz with tight timing and cultural references. Supports live Selenium runs, offline replay, and provider abstractions.

Key Features

  • Live runs via browser automation
  • Replayable datasets & JSON artifacts
  • Provider abstractions: OpenAI-compatible and Gemini
  • Cost & token usage reporting per run
  • OpenCode GitHub automations for PRs, triage, and scheduled sweeps

Quickstart

uv sync --group dev then run a replay benchmark:

uv run duel benchmark --source replay --dataset examples/replay_sample.json --provider baseline --runs 2
