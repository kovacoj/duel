# Contributor Covenant Code of Conduct

Our project follows the Contributor Covenant v2.1. By participating, you are
expected to uphold this code. The full text is reproduced here (shortened).

Be respectful, inclusive, and considerate. Harassment, discrimination, and
abusive behavior are not acceptable. Please report issues to the maintainers
or open an issue marked `security` for sensitive concerns.

For the complete policy and reporting instructions, maintainers will follow
the standard Contributor Covenant process.
